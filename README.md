# normalize-google-docs
## 概要
Googleドキュメント内に結合文字列が存在した場合、合成文字に置換するスクリプト  
例：「か゛」 -> 「が」   
## 開発手順
このリポジトリをローカルにclone  
```
yarn start
```
ファイルを保存するとGASに反映される
